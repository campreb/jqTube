jQuery plugin for customizable YouTube controls.

Dependencies

  • jQuery 1.3.2
  • jQuery UI 1.7.2
  • swfloader 2.2.2

Contact

Cameron Prebble (cameron@projectx.co.nz)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/campreb/juTube

get the source code on GitHub : campreb/juTube
